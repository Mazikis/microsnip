Microsnip

Microsnip is a collection of code snippets for CMS and frameworks.

The idea behind this project is help others and myself to find pieces of information on one page rather than trying to Google every little thing.

Code snippets should not be longer than a few lines and I will always try to credit authors (if there are any).
